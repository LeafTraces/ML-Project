# 三种方法的理论细节（LR / SVM / U-Net）

> 用于答辩/报告的"方法理论"部分。每个公式都给了可读写法；关键公式额外附 LaTeX 代码块，方便直接粘进 Word(MathType) 或 LaTeX。每节末尾「与本任务的联系」把理论和你的结果挂钩。

设每个像素的特征向量为 **x ∈ ℝ¹⁴**（14 维手工特征），标签 y（渗出=1，背景=0）。

---

## 一、逻辑回归（Logistic Regression, LR）

### 1.1 模型形式
LR 先对特征做线性打分（logit）：
$$z = \mathbf{w}^\top \mathbf{x} + b$$
再用 Sigmoid 把分数压到 (0,1) 作为"是渗出"的概率：
$$P(y=1\mid \mathbf{x}) = \sigma(z) = \frac{1}{1+e^{-z}}$$

```latex
P(y=1\mid \mathbf{x})=\sigma(\mathbf{w}^\top\mathbf{x}+b)=\frac{1}{1+e^{-(\mathbf{w}^\top\mathbf{x}+b)}}
```

### 1.2 几何与可解释性
对概率取 logit 可得**对数几率（log-odds）关于特征线性**：
$$\ln\frac{P(y=1\mid\mathbf{x})}{P(y=0\mid\mathbf{x})} = \mathbf{w}^\top\mathbf{x}+b$$
- 决策边界 $\mathbf{w}^\top\mathbf{x}+b=0$ 是特征空间里的一张**超平面**（线性）。
- 权重 $w_j$ 的含义：特征 $x_j$ 每增加一个单位，对数几率改变 $w_j$ → **系数即特征重要性**（这正是 fig2 的来源）。

### 1.3 目标函数（极大似然 = 交叉熵）
对 N 个像素，Bernoulli 似然取负对数得交叉熵损失，加 L2 正则与类别权重：
$$\mathcal{L}(\mathbf{w},b)=\frac12\lVert\mathbf{w}\rVert^2 + C\sum_{i=1}^{N}\omega_{y_i}\Big[-y_i\ln p_i-(1-y_i)\ln(1-p_i)\Big],\quad p_i=\sigma(\mathbf{w}^\top\mathbf{x}_i+b)$$

```latex
\mathcal{L}(\mathbf{w},b)=\tfrac12\lVert\mathbf{w}\rVert^2
+C\sum_{i=1}^{N}\omega_{y_i}\big[-y_i\ln p_i-(1-y_i)\ln(1-p_i)\big]
```
- $C$ 是正则强度的倒数（我们取 $C=1$）；$\omega_{y_i}$ 是类别权重（`class_weight=balanced`，与类频率成反比，缓解不平衡）。
- 该目标是**凸函数** → 有全局最优，用 lbfgs/牛顿法可靠求解。

### 1.4 梯度（便于讲优化）
对单样本，交叉熵对 $\mathbf{w}$ 的梯度形式非常简洁：
$$\frac{\partial \mathcal{L}_i}{\partial \mathbf{w}} = (p_i - y_i)\,\mathbf{x}_i$$
即"预测概率与真实标签之差"乘以特征——和线性回归梯度同形，只是 $p_i$ 经过了 Sigmoid。

### 1.5 与本任务的联系
LR 是**线性**模型，决策面是超平面；它无法表达特征间的非线性交互。fig2 显示其权重集中在 L(亮度)、b(黄度) 等颜色通道——说明它本质上在用"够不够亮、够不够黄"来线性划界，因而把**所有亮黄像素**（含血管反光、视盘边缘）都判为渗出，导致**精确率极低（0.048）、误报近 10 万**。

## 二、支持向量机（Support Vector Machine, SVM）

标签记为 $y\in\{-1,+1\}$。

### 2.1 最大间隔思想
SVM 寻找一张**间隔最大**的分离超平面 $\mathbf{w}^\top\mathbf{x}+b=0$。样本到超平面的几何间隔为
$$\gamma_i = \frac{y_i(\mathbf{w}^\top\mathbf{x}_i+b)}{\lVert\mathbf{w}\rVert}$$
最大化最小间隔，等价于在"函数间隔≥1"约束下最小化 $\lVert\mathbf{w}\rVert$。

### 2.2 软间隔原问题（primal）
真实数据不可完全线性可分，引入松弛变量 $\xi_i\ge0$：
$$\min_{\mathbf{w},b,\boldsymbol\xi}\ \frac12\lVert\mathbf{w}\rVert^2 + C\sum_i \xi_i
\quad\text{s.t.}\quad y_i(\mathbf{w}^\top\mathbf{x}_i+b)\ge 1-\xi_i,\ \ \xi_i\ge0$$

```latex
\min_{\mathbf{w},b,\boldsymbol\xi}\ \tfrac12\lVert\mathbf{w}\rVert^2+C\sum_i\xi_i
\quad\text{s.t.}\quad y_i(\mathbf{w}^\top\mathbf{x}_i+b)\ge1-\xi_i,\ \xi_i\ge0
```
- $C$ 权衡"间隔大"与"分类错误少"。

### 2.3 等价的合页损失（hinge loss）形式
把约束代入，可写成无约束的"合页损失 + 正则"：
$$\min_{\mathbf{w},b}\ \frac12\lVert\mathbf{w}\rVert^2 + C\sum_i \max\big(0,\,1-y_i(\mathbf{w}^\top\mathbf{x}_i+b)\big)$$
合页损失 $\max(0,1-y f(x))$：分类正确且置信（间隔≥1）时损失为 0，否则线性惩罚。

### 2.4 对偶问题与支持向量
引入拉格朗日乘子 $\alpha_i\ge0$，得对偶问题：
$$\max_{\boldsymbol\alpha}\ \sum_i\alpha_i-\frac12\sum_{i,j}\alpha_i\alpha_j y_i y_j\,\mathbf{x}_i^\top\mathbf{x}_j
\quad\text{s.t.}\ \ 0\le\alpha_i\le C,\ \ \sum_i\alpha_i y_i=0$$
解出后 $\mathbf{w}=\sum_i\alpha_i y_i\mathbf{x}_i$。只有落在间隔边界上/内的样本 $\alpha_i>0$，称为**支持向量**——决策只由它们决定。

### 2.5 核技巧与 RBF 核
对偶问题**只依赖样本内积** $\mathbf{x}_i^\top\mathbf{x}_j$。用核函数 $K(\mathbf{x},\mathbf{x}')=\phi(\mathbf{x})^\top\phi(\mathbf{x}')$ 替换内积，就能在**隐式的高维空间** $\phi$ 里做线性划分（即原空间的非线性边界），而无需显式计算 $\phi$。**高斯/RBF 核**：
$$K(\mathbf{x},\mathbf{x}')=\exp\!\big(-\gamma\lVert\mathbf{x}-\mathbf{x}'\rVert^2\big)$$

```latex
K(\mathbf{x},\mathbf{x}')=\exp\!\big(-\gamma\lVert\mathbf{x}-\mathbf{x}'\rVert^{2}\big)
```
对应**无穷维**特征空间；$\gamma$ 控制"作用半径"：$\gamma$ 大→边界越尖锐、易过拟合；$\gamma$ 小→越平滑。判别函数：
$$f(\mathbf{x})=\sum_i\alpha_i y_i K(\mathbf{x}_i,\mathbf{x})+b$$

### 2.6 为什么像素级要做核近似（本实验的关键工程点）
精确核 SVM：训练约 $O(n^2\!\sim\! n^3)$，预测约 $O(n_{SV}\times n)$。本任务训练/预测涉及**数百万像素**，支持向量上万 → 直接做会卡死（我们前期实测如此）。

**Nystroem 核近似**：从训练样本里取 $m$ 个地标 $\{\mathbf{z}_j\}_{j=1}^m$（我们取 $m=200$），用地标间的核矩阵 $W_{jk}=K(\mathbf{z}_j,\mathbf{z}_k)$ 构造**显式**特征映射
$$\hat\phi(\mathbf{x}) = W^{-1/2}\,\big[K(\mathbf{x},\mathbf{z}_1),\dots,K(\mathbf{x},\mathbf{z}_m)\big]^\top \in\mathbb{R}^{m}$$
使 $\hat\phi(\mathbf{x})^\top\hat\phi(\mathbf{x}')\approx K(\mathbf{x},\mathbf{x}')$（本质是对核 Gram 矩阵做低秩逼近）。映射后**接线性 SVM**，训练/预测降到约 $O(n\cdot m)$，像素级可扩展。

```latex
\hat\phi(\mathbf{x})=W^{-1/2}\big[K(\mathbf{x},\mathbf{z}_1),\dots,K(\mathbf{x},\mathbf{z}_m)\big]^\top
```

### 2.7 线性 SVM 的 SGD 求解
在 $\hat\phi$ 空间用随机梯度下降最小化（平滑的）合页损失。我们用 `modified_huber` 损失——合页损失的二次平滑变体，可导且能给出概率估计：
$$\ell(f,y)=\begin{cases}\max(0,1-yf)^2,& yf\ge-1\\ -4yf,& yf<-1\end{cases}$$

### 2.8 与本任务的联系
RBF 核理论上能拟合非线性边界；但经 Nystroem(200) 近似 + 未精调 $\gamma/C$，其非线性增益未充分体现，本实验中**略逊于 LR**（Dice 0.074 vs 0.086）。且它和 LR 一样只用**局部像素特征**，无法利用空间上下文 → 误报仍多（6.7 万）。

## 三、U-Net（全卷积分割网络）

### 3.1 卷积与感受野（为什么 CNN 适合图像）
卷积层用一个小核 $K$ 在特征图上滑动做局部加权和：
$$(I * K)(i,j)=\sum_{m}\sum_{n} I(i+m,\,j+n)\,K(m,n)$$
- **权值共享 + 局部连接** → 平移等变、参数少。
- 随着层数加深 + 下采样，每个输出像素的**感受野**（它"看得到"的输入区域）不断扩大 → 高层神经元能利用**大范围上下文**做判断。这是 U-Net 能区分"真渗出 vs 像渗出的反光"的根本原因，也是它对像素级 LR/SVM 的本质优势。

每个卷积块通常是 Conv → BatchNorm → ReLU：
- **BatchNorm**：对一批激活做标准化，稳定并加速训练。
- **ReLU** $\max(0,z)$：引入非线性。

### 3.2 编码器–解码器 + 跳跃连接
U-Net 是对称的"U"形全卷积网络：
- **编码器（下采样路径）**：重复 卷积块 + 池化/步长卷积，空间分辨率逐级减半、通道数逐级加倍 → 提取由浅到深的语义、扩大感受野（"看懂是什么"）。
- **解码器（上采样路径）**：用**转置卷积**或插值逐级上采样，恢复分辨率（"定位在哪里"）。
- **跳跃连接（skip connection）**：把编码器同尺度的特征图**拼接**到解码器对应层。下采样会丢失精细的空间细节，跳跃连接把这些细节直接补回来 → 对**小而散的病灶**边界定位至关重要。
- **输出头**：$1\times1$ 卷积 → 每像素一个 logit → Sigmoid → 概率图。

### 3.3 编码器：ResNet34 与迁移学习
编码器采用在 ImageNet 上**预训练**的 ResNet34。残差块的核心是**恒等捷径**：
$$\mathbf{y}=\mathcal{F}(\mathbf{x},\{W_i\})+\mathbf{x}$$
让网络学"残差" $\mathcal{F}$，缓解深层网络的梯度消失/退化问题。**迁移学习**：低层卷积学到的边缘/纹理等通用特征可迁移到眼底图，这是仅 47 张小样本仍能训练好的关键（从零训练会严重欠拟合，我们前期已验证）。

### 3.4 损失函数（对抗极端类别不平衡）
- **加权 BCE**（逐像素交叉熵，正样本加权 $w_p$）：
$$\mathcal{L}_{BCE}=-\frac1N\sum_i\big[w_p\,y_i\ln p_i+(1-y_i)\ln(1-p_i)\big]$$
- **Dice 损失**（直接优化重叠度，不计真阴性 → 抗不平衡）：
$$\mathcal{L}_{Dice}=1-\frac{2\sum_i p_i g_i+\varepsilon}{\sum_i p_i+\sum_i g_i+\varepsilon}$$
- **Tversky 损失**（Dice 的推广，可调假阳/假阴权重）：
$$TI=\frac{\sum_i p_i g_i}{\sum_i p_i g_i+\alpha\sum_i p_i(1-g_i)+\beta\sum_i(1-p_i)g_i},\quad \mathcal{L}_{Tv}=1-TI$$
其中 $g_i$ 为真值、$p_i$ 为预测概率。$\alpha=\beta=0.5$ 时退化为 Dice；我们取 $\boldsymbol{\alpha=0.3,\ \beta=0.7}$，即**更重罚假阴（漏检）**，从而提升小病灶召回。
- **我们的组合损失**：
$$\mathcal{L}=w\cdot\mathcal{L}_{BCE}+(1-w)\cdot\mathcal{L}_{Tv}$$

```latex
TI=\frac{\sum_i p_i g_i}{\sum_i p_i g_i+\alpha\sum_i p_i(1-g_i)+\beta\sum_i (1-p_i)g_i},\qquad
\mathcal{L}=w\,\mathcal{L}_{BCE}+(1-w)\,(1-TI)
```

### 3.5 优化与训练技巧
- **AdamW**：带"解耦权重衰减"的 Adam。维护一阶/二阶矩估计并做偏差校正，参数更新：
$$\mathbf{m}_t=\beta_1\mathbf{m}_{t-1}+(1-\beta_1)\mathbf{g}_t,\quad \mathbf{v}_t=\beta_2\mathbf{v}_{t-1}+(1-\beta_2)\mathbf{g}_t^2$$
$$\theta_{t}=\theta_{t-1}-\eta\Big(\frac{\hat{\mathbf{m}}_t}{\sqrt{\hat{\mathbf{v}}_t}+\epsilon}+\lambda\,\theta_{t-1}\Big)$$
- **余弦退火学习率**：让学习率沿余弦曲线从 $\eta_{\max}$ 平滑降到 $\eta_{\min}$：
$$\eta_t=\eta_{\min}+\tfrac12(\eta_{\max}-\eta_{\min})\Big(1+\cos\frac{t}{T}\pi\Big)$$
- **混合精度（AMP）**：用 fp16/bf16 计算、fp32 保存主权重 → 更快、更省显存。
- **数据增强**（翻转 / 旋转 / 亮度抖动）：对小样本是有效的正则化，缓解过拟合。
- **模型选择**：按验证集 Dice 取最佳轮权重（早停思想）。

### 3.6 与本任务的联系
跳跃连接 + 大感受野让 U-Net 既"看懂上下文"又"精确定位"，因而**假阳性骤降到 4,383**（LR/SVM 为 6.7–9.9 万），精确率 0.536 远超传统法的 ~0.05，Dice 达 0.480。fig3 训练曲线显示其稳定收敛、无明显过拟合，印证了"预训练编码器 + 组合损失 + 增强"这套设计的有效性。

## 四、三者理论对比（可做一页小结）

| 维度 | 逻辑回归 LR | SVM (RBF/Nystroem) | U-Net |
|---|---|---|---|
| 决策边界 | 线性超平面 | 非线性（核诱导） | 高度非线性（深层卷积） |
| 用到的信息 | 单像素局部特征 | 单像素局部特征 | **空间上下文 + 形状**（感受野） |
| 目标函数 | 交叉熵 + L2 | 合页损失 + 间隔最大化 | 加权 BCE + Tversky |
| 优化 | 凸、lbfgs 全局最优 | 凸（对偶 QP）/ SGD | 非凸、AdamW + 余弦退火 |
| 可解释性 | 高（看系数） | 中（支持向量/核） | 低（黑箱） |
| 关键超参 | C、类别权重 | C、γ、地标数 m | 编码器、α/β、lr、轮数 |
| 本任务表现 | Dice 0.086 | Dice 0.074 | **Dice 0.480** |

**一句话理论总结**：LR/SVM 是"逐像素 + 手工特征"的分类器，受限于局部信息，对"亮黄但非渗出"的结构无能为力；U-Net 通过卷积感受野引入**空间上下文**，并用预训练迁移与抗不平衡损失，从根本上压低了误报——这正是它在该任务上全面领先的理论原因。
